## Success Criteria

- [ ] Submit prompt → receive streamed tokens via SSE
- [ ] 3 concurrent requests queue correctly (only 1 runs at a time on GPU)
- [ ] Model not in memory → auto-loaded before inference
- [ ] Memory full → LRU eviction (non-active model only)
- [ ] `docker compose up` works out of the box
- [ ] `OBSERVABILITY_BACKEND=stdout|otel|clickhouse` switches adapter

## Constraints

- Single GPU (APU, 96GB unified memory)
- `OLLAMA_KEEP_ALIVE=-1` (greedy allocation)
- ARQ `max_jobs=1` (serial GPU processing)
- SSE behind Cilium Gateway: `proxy_buffering off` required
