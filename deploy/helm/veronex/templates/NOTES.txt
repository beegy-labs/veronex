{{- else }}

  Get the dashboard URL:
    kubectl port-forward svc/{{ .Release.Name }}-veronex-web {{ .Values.veronexWeb.service.port }}:{{ .Values.veronexWeb.service.port }}
    open http://localhost:{{ .Values.veronexWeb.service.port }}

  Get the API URL:
    kubectl port-forward svc/{{ .Release.Name }}-veronex {{ .Values.veronex.service.port }}:{{ .Values.veronex.service.port }}
    curl http://localhost:{{ .Values.veronex.service.port }}/health

{{- end }}

Components:
  - veronex (API server + scheduler)
  - veronex-analytics (OTel ingest + read API)
  - veronex-web (Next.js dashboard)
{{- if .Values.otelCollector.enabled }}
  - otel-collector (OpenTelemetry Collector)
{{- end }}

First run:
  1. Visit the dashboard and complete the setup wizard
  2. Add an Ollama provider URL in Dashboard > Providers
  3. Create an API key in Dashboard > Keys
  4. Test: curl http://<api>/v1/chat/completions -H "Authorization: Bearer iq_..."
